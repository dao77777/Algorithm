- Master公式: T(n) = a * T(n/b) + O(n^d), 子问题规模一致时可用此公式求时间复杂度
  - log(b)(a) < d: O(n^d)
  - log(b)(a) > d: O(n^log(b)(a))
  - log(b)(a) = d: O(n^d * logn)
- 排序
  - 选择排序: 
    - 时间复杂度: O(n^2)
    - 空间复杂度: O(1)
  - 冒泡排序: 
    - 时间复杂度: O(n^2)
    - 空间复杂度: O(1)
  - 插入排序: 
    - 时间复杂度: O(n^2)
    - 空间复杂度: O(1)
  - 归并排序: 
    - 时间复杂度: O(nlogn)
    - 空间复杂度: O(n)
  - 快速排序
    - 时间复杂度: O(nlogn)
    - 空间复杂度: 平均O(logn), 最差O(n)
  - 堆排序
    - 时间复杂度: O(nlogn)
    - 空间复杂度: O(1)
