- Master公式: T(n) = a * T(n/b) + O(n^d), 子问题规模一致时可用此公式求时间复杂度
  - log(b)(a) < d: O(n^d)
  - log(b)(a) > d: O(n^log(b)(a))
  - log(b)(a) = d: O(n^d * logn)
- 排序
  - 选择排序: O(n^2), O(1)
  - 冒泡排序: O(n^2), O(1)
  - 插入排序: O(n^2), O(1)
  - 归并排序: O(nlogn), O(n)
  - 快速排序: O(nlogn), O(1)
